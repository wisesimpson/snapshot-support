[English](#english) | [中文](#中文)

## English

This is the support page for Snapshot.

Snapshot is a Safari extension for capturing clean, high-quality web page screenshots.

### Get Help

If you have questions, found a bug, or want to request a feature, please open an issue in this repository.

### Features

- Preset window sizes
- Custom dimensions
- Timer capture
- PNG export

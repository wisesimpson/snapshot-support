# Snapshot Support

[English](#english) | [中文](#中文)

## English

This is the support page for Snapshot, a Safari extension for capturing clean, high-quality web page screenshots.

### Get Help

If you have questions, found a bug, or want to request a feature, please open an issue in this repository.

### About Snapshot

Snapshot helps you capture web pages in Safari with:

- Preset window sizes
- Custom dimensions
- Timer capture
- PNG export

## 中文

这里是 Snapshot 的支持页面。Snapshot 是一款 Safari 扩展，用于快速截取清晰、高质量的网页截图。

### 获取帮助

如果你在使用过程中遇到问题、发现 Bug，或者希望提出功能建议，欢迎在此仓库提交 Issue。

### 关于 Snapshot

Snapshot 可以帮助你在 Safari 中：

- 使用预设窗口尺寸截图
- 使用自定义尺寸截图
- 进行定时截图
- 导出 PNG 图片
